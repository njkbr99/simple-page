  • Мати веб-сторінку, яку інші зможуть побачити в інтернеті

Докладна інструкція:

  1. Підготуй обліковий запис GitHub

    1. Перейди на ➜ github.com (сайт відкриється у новій вкладинці твого браузера якщо ти натиснеш на посилання ліворуч).
    2. Натисни кнопку Sign up у верхньому правому куті.
    3. Введи e‑mail, створи пароль, ім'я користувача (англійською), обери країну та натисни Create account.
      Це приклад як я створював акаунт, в тебе буде інший email, username, та інше:
    4. Тебе попросять ввести код, який ти отримаєш на свій email:
    5. Введи код, який тобі прислали
    6. Тебе попросять авторизуватись з тим email і паролем, який ти використав щоб створити акаунт:

  2. Створи новий репозиторій (місце де будуть зберігатися твої файли) для сторінки

    1. У верхній панелі GitHub натисни піктограму + ➜ New repository (верхній правий кут).
    2. У полі Repository name введи назву репозиторію, можеш придумати сам, або написати <simple-page>
    3. Переконайся, що вибрано радіокнопку Public.
    4. Натисни Create repository.

  3. Додай файл index.html

    1. Відкрий сторінку свого проєкту.
      1. Натисни на іконку меню у верхньому лівому куті.
      2. Натисни на назву свого проєкету.
    2. На сторінці нового репозиторію натисни посилання creating new file.
    3. У полі file name введи index.html.
    4. Напиши будь-який текст.
    5. У правій частині екрану буде зелена кнопка Commit changes... - натисни її.
    6. Натисни Commit changes у вікні яке зʼявиться.

  4. Увімкни GitHub Pages

    1. Перейди на вкладку Settings (праворуч, під назвою репозиторію).
    2. У лівому меню прокрути униз і натисни Pages.
    3. У секції Build and deployment → Source вибери:
      • Deploy from a branch.
      • Branch = main.
      • Folder = / (root).
      Після цього натисни Save.
    4. Зачекай 5 хвилин.
    5. Перезавантаж сторінку.
    6. Вище з'явиться плашка: «Your site is live at …». Натисни на посилання.

  5. Перевір сторінку

    1. Через 30‑120 секунд має з'явитися твоя сторінка з текстом, який ти додав.

  6. Корисні поради

    1. Якщо бачиш 404, зачекай ще хвилину та онови сторінку з Ctrl + F5.
    2. Щоб змінити вміст, редагуй index.html у репозиторії й тисни Commit changes — оновлення з'являться автоматично.
